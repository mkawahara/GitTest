このフォルダには、jsSHA のアーカイブが記録されています。
ダウンロード元は、http://caligatio.github.io/jsSHA/

最新版の v2.0.1 を使うと、ChattyInftyOnline のログイン時に
"Chosen SHA variant is not supported" という JavaScript エラーが発生します。
そのため、v1.6.1 を使用しています。

アーカイブ内の src/sha256.js だけを、
ChattyInftyOnline の www/js/sha256.js にコピーして使用しています。
